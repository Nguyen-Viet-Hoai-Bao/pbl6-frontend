# Installation
1. Clone the repository

```
git clone
```

2. Install dependencies

```
cd folder_name
npm install
```

3. Configure environment variables
- In the root create .env file and configure the following variables

```
MONGO_URI=
GITHUB_ID=
GITHUB_SECRET=
NEXTAUTH_SECRET=
NEXTAUTH_URL=
GOOGLE_ID=
GOOGLE_SECRET=
```

4. Run the application

```
npm run dev
```
